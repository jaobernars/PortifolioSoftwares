Project 3 - Dashboard de Finanças Pessoais

O que faz: Uma aplicação web simples onde o usuário insere seus gastos diários por categoria e o programa gera gráficos visuais de pizza ou de barras mostrando para onde o dinheiro está indo.

O que você aprende: Criação de interfaces web rápidas com a biblioteca Streamlit (que é fantástica e super simples para iniciantes) e geração de gráficos com Plotly ou Matplotlib.
