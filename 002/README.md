Project 2 - Monitor de Preços ou Clima com Notificação

O que faz: O programa consulta uma API gratuita (como uma API de clima ou de cotação de moedas), extrai a informação e, se algo atingir um limite configurado (ex: temperatura menor que 15°C ou dólar abaixo de um valor), ele te envia um e-mail automático ou uma mensagem no Telegram.

O que você aprende: Como fazer requisições HTTP (com a biblioteca requests), ler dados em formato JSON e enviar e-mails ou integrar com bots usando Python.
