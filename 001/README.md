Project 1 - Organizador Automático de Pastas

O que faz: Um script que varre uma pasta bagunçada (como a sua pasta de "Downloads") e move automaticamente arquivos para pastas específicas baseado na extensão (ex: .pdf vai para a pasta "Documentos", .png e .jpg vão para "Imagens").

O que você aprende: Manipulação de caminhos do sistema operacional (usando a biblioteca padrão os ou pathlib).
